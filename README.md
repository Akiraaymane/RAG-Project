# RAG Project with LangChain and Groq

A simple **Retrieval-Augmented Generation (RAG)** pipeline using LangChain, Chroma vector store, and Groq LLM.  
This project loads documents, creates embeddings, retrieves relevant content, and answers user queries.

---

## Features

- Load and split documents from a directory.
- Index documents with embeddings (supports incremental indexing).
- Retrieve relevant chunks for a query.
- Generate answers using **ChatGroq LLM**.
- Configurable through `config.yaml`.
